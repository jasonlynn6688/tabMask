# Tab Mask · 标签页重命名

> 把 YouTube 改名叫「项目文档」，摸鱼从此光明正大。

---

## 这是什么

一个 Chrome 插件，让你给浏览器标签页起个一本正经的名字。

老板走过来？标签页上写着「工作台」，稳得很。

## 功能

- **一键改名** — 打开插件，选个名字，应用，完事
- **智能建议** — 自动推荐几个听起来很忙的名字
- **历史记录** — 上次用过的名字，下次一键复用
- **恢复原名** — 下班了，一键还原

## 安装

目前需要手动加载（Chrome 应用商店上架中）：

1. 下载或 clone 本仓库
2. 打开 Chrome，进入 `chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择 `src/` 目录

## 使用

1. 在任意普通网页点击扩展图标
2. 从建议中挑一个，或者自己输入
3. 点「应用更改」
4. 弹窗关闭，标签页已改名 ✓

想恢复原名？再打开插件点「恢复默认」。

## 支持的页面

普通 `http` / `https` 网页均支持。Chrome 内置页（`chrome://`）和应用商店页面不支持。
